Info! We will be programming a simple robot. The robot is composed of a number of blocks and reacts to a simple set of commands, written in a program.
A program is composed of:

- An integer, the number of blocks the robot contains

- A series of commands, one per line.The commands should be like :move A onto B AND move A over B
Exemple:
3,
move 2 onto 0,
move 1 over 0,
quit

- A quit instruction

Note: After each command you have to use ' , ' as separator.
